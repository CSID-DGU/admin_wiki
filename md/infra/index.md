# Infra

## 개요

Infra 영역은 Kubernetes, 인증, 스토리지처럼 사용자 실행 환경의 기반이 되는
공용 인프라를 다룬다. 애플리케이션 기능과 별개로 계정 신원, 자격 증명, 노드와
공유 스토리지의 경계를 안전하게 유지하는 것이 목적이다.
